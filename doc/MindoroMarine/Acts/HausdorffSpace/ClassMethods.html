roots() click to toggle source

Get all the top level roots. This does a find of all records with no immediate parent in the database. When they’re loaded they get a virtual root attached as their parent. This is to make sure that every node has a parent which helps to generalise and simplify the code for working out left and right boundaries. 

# File lib/acts_as_hausdorff_space.rb, line 85
    def roots
     top_levels =    self.find( :all, :conditions => "not exists (select t1.id from #{self.table_name} t1
                                                                       where t1.#{left_col_name} <#{self.table_name}.#{left_col_name} 
                                                                       and t1.#{right_col_name} > #{self.table_name}.#{right_col_name})",:order=>"#{left_col_name}")
     self.virtual_root.children.clear
     top_levels.each do |tl|
     begin
     tl.in_tree = true
     self.virtual_root.children << tl
     tl.in_tree = false
     rescue
     tl.in_tree = false
     raise
     end
     end 
    end
